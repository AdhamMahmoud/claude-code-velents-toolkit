# PM Strategy Frameworks (B2B SaaS)

## Lean Canvas — AI Agent Platform

| Block | VelentsAI Context |
|-------|------------------|
| **Problem** | Manual screening is slow, biased, and doesn't scale; HR lacks AI tools they can configure without engineering |
| **Solution** | Configurable AI agents for voice/text conversations with candidates, multi-channel deployment |
| **UVP** | "Deploy AI interview agents in minutes, no code required" |
| **Unfair Advantage** | Multi-tenant architecture, Arabic + English NLP, voice pipeline with 3 speed tiers |
| **Customer Segments** | Enterprise HR (500+ employees), staffing agencies, BPO companies |
| **Key Metrics** | ARR, NRR, agents deployed per tenant, conversations/month, activation rate |
| **Channels** | Direct sales, partnerships, product-led (self-serve trial), content marketing |
| **Cost Structure** | Cloud infrastructure, AI API costs (LLM, TTS, STT), engineering, customer success |
| **Revenue Streams** | Per-agent subscription, credit-based usage (calls/messages), enterprise licensing |

## Porter's Five Forces — AI Agent Market

| Force | Assessment | Notes |
|-------|-----------|-------|
| **New Entrants** | HIGH (4/5) | Low capital barriers, many AI startups, but moats via integrations and data |
| **Supplier Power** | MEDIUM (3/5) | Dependent on LLM providers (OpenAI, Google) but multi-provider strategy helps |
| **Buyer Power** | MEDIUM (3/5) | Enterprise buyers negotiate hard, but switching costs increase with usage |
| **Substitutes** | MEDIUM (3/5) | Manual processes, in-house AI teams, generic chatbot platforms |
| **Rivalry** | HIGH (4/5) | Crowded AI space, but niche (HR AI agents) is less contested |

## B2B SaaS Metrics

| Metric | Formula | Good Benchmark |
|--------|---------|---------------|
| **ARR** | MRR × 12 | Growing 2-3x YoY (early stage) |
| **NRR (Net Revenue Retention)** | (Start MRR + Expansion - Contraction - Churn) / Start MRR | >110% (great), >120% (excellent) |
| **NDR (Net Dollar Retention)** | Same as NRR, industry-interchangeable | >100% means growth without new sales |
| **Gross Churn** | Lost MRR / Start MRR | <5% monthly, <10% annual |
| **CAC** | Total S&M spend / New customers acquired | Varies by segment |
| **CAC Payback** | CAC / (ARPU × Gross Margin) | <12 months (good), <6 months (great) |
| **LTV** | ARPU × Gross Margin × (1 / Churn Rate) | LTV:CAC > 3:1 |
| **Magic Number** | Net New ARR / S&M Spend (prior quarter) | >0.75 = invest more in sales |
| **Rule of 40** | Revenue Growth Rate + Profit Margin | >40% combined |

## Platform Strategy — Multi-Sided Considerations

### Platform Flywheel for VelentsAI
```
More Tenants → More Conversations → Better AI Models → Better Outcomes →
More Tenants (network effect)
```

### Platform Expansion Vectors
| Vector | Description | Metric |
|--------|-------------|--------|
| **Depth** | More agents per tenant, more conversations | Agents/tenant, conversations/agent |
| **Breadth** | New use cases beyond HR (sales, support, onboarding) | % revenue from non-HR |
| **Ecosystem** | Integrations, marketplace, partner channels | # integrations, partner-sourced ARR |
| **Geographic** | MENA → Global expansion | % revenue by region |
